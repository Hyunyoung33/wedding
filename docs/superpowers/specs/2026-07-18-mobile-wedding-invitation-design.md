# 모바일 청첩장 설계 문서

- 작성일: 2026-07-18
- 신랑·신부: 두현민 · 박현영
- 상태: 사용자 승인 완료 (아키텍처)

## 1. 목표

시중 유료 모바일 청첩장(필프레임 풀옵션 기준)과 동등한 기능의 청첩장 웹사이트를 직접 만든다.

핵심 요구사항 3가지:

1. **즉시 열림** — 링크를 카카오톡 등으로 보내면 앱 설치·로그인 없이 바로 열린다.
2. **상시 수정 가능** — 문구·날짜·사진을 언제든 Claude Code에게 요청해 1~2분 내 수정·재배포한다.
3. **영구 보존** — 유료 청첩장처럼 몇 개월 뒤 만료되지 않는다. 무료로 계속 유지된다.

## 2. 아키텍처

- **프론트엔드**: 정적 웹사이트 (HTML/CSS/JS). 빌드 도구는 Vite 사용, 결과물은 순수 정적 파일.
- **호스팅**: GitHub Pages (무료, 기간 제한 없음). URL은 `<계정>.github.io/wedding` 형태.
- **동적 데이터 (방명록·RSVP)**: Firebase Firestore 무료(Spark) 플랜. 서버리스, 자동 일시정지 없음.
- **지도**: 카카오맵 JavaScript SDK (무료 키).
- **카카오톡 공유**: 카카오 JavaScript SDK + Open Graph 메타태그.
- **콘텐츠 관리**: 모든 문구·일시·계좌·연락처를 단일 설정 파일(`src/data/wedding.js`)에 집중. 수정 = 이 파일 한 곳만 변경.

### 대안 검토 (제외 사유)

- Vercel/Netlify + Supabase: Supabase 무료 플랜은 1주 무접속 시 DB 일시정지 → 영구 보존 요구에 부적합.
- 유료 청첩장 구매: 수정 시 업체 경유, 링크 수개월 후 만료 → 요구사항과 정면 충돌.

## 3. 기능 목록

### 정적 (서버 불필요)

1. 인트로 애니메이션 + 메인 커버 (사진, 이름, 일시)
2. 인사말(모시는 글)
3. 신랑·신부·혼주 소개 및 전화/문자 버튼
4. 달력 하이라이트 + D-day 카운트다운
5. 사진 갤러리 (스와이프, 라이트박스 확대)
6. 오시는 길 — 카카오맵, 주소 복사, 카카오내비/티맵/네이버지도 딥링크, 대중교통·주차 안내
7. 계좌번호 — 아코디언, 계좌 복사, (선택) 카카오페이 송금 링크
8. 배경음악 토글 (모바일 자동재생 제약 준수)
9. 카카오톡 공유 커스텀 카드 + URL 복사
10. (선택) 식전 영상 유튜브 임베드

### 동적 (Firebase)

11. 방명록 — 이름+메시지 작성, 목록 실시간 표시, 간단한 스팸 방지(글자수 제한, 속도 제한)
12. RSVP — 참석 여부/동행 인원/식사 여부 수집. 신랑신부 전용 집계 확인 방법 제공(Firebase 콘솔 또는 비밀 파라미터 페이지)

## 4. 데이터 흐름

- 하객 → 정적 페이지 로드(GitHub Pages) → 방명록/RSVP 작성 시 브라우저에서 Firestore로 직접 쓰기.
- Firestore 보안 규칙: 방명록/RSVP 컬렉션은 생성(create)만 허용, 수정·삭제는 소유자(콘솔)만. 읽기는 방명록만 공개.

## 5. 에러 처리

- Firebase 연결 실패 시: 방명록/RSVP 섹션만 "잠시 후 다시 시도해주세요" 표시, 나머지 청첩장은 정상 동작 (graceful degradation).
- 카카오맵 로드 실패 시: 정적 지도 이미지 + 주소 텍스트로 대체.
- 이미지: lazy loading + 모바일 최적화(리사이즈·압축, WebP)로 3G에서도 열리게.

## 6. 테스트·검증

- 모바일 뷰포트(375px) 기준 브라우저 확인, 카카오톡 인앱 브라우저 테스트.
- 공유 카드(OG 태그) 카카오톡 미리보기 검증.
- 방명록/RSVP 실제 쓰기 테스트 후 Firestore 콘솔에서 확인.

## 7. 사용자가 준비할 것 (각 5분 내외, 진행 시 안내)

1. GitHub 저장소 생성 권한 (계정 보유 확인됨)
2. Firebase 프로젝트 생성 (구글 계정)
3. 카카오 개발자 앱 키 (카카오 계정)
4. 사진 선별: 메인 1장 + 갤러리 사진 (웨딩촬영 폴더에서)
5. 문구: 인사말, 혼주 성함, 예식장 정보, 계좌번호, 연락처
